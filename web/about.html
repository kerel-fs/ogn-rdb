OGN list of receivers - About

The OpenGliderNetwork is a centralized network focused on tracking of aircrafts equipped with FLARM, FLARM-compatible devices or OGN tracker. This page shows all registered OGN receivers.

The displayed data is parsed from the wiki page list-of-receivers and updated hourly.
Live receiver status is provided by the aprsc servers directely and historical uptime statistics are provided by OGNRange.

For now, email addresses are stripped to prevent spam.

Data sources

Receiver database

Data source: list-of-receivers
Data file: receivers.json (latest version, updated hourly)
File format: receiverlist-schema-0.2.1.json

APRSC Server Status

Data source: glidern1 and glidern2
Data file: glidern1/status.json and glidern2/status.json

OGNRange

Data source: OGNRange
Data format: OGNRange API

Privacy Stats

Data file: receivers-privacy.json
